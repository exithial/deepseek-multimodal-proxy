🎯 **DeepSeek Multimodal Proxy v1.8.0**

La versión `1.8.0` introduce dos mejoras operativas grandes: compatibilidad real con Windows y despliegue persistente con Docker Compose.

## 🪟 **Soporte nativo para Windows**

- Scripts `setup`, `manage` y `run-local` ahora tienen implementación PowerShell
- `package.json` usa comandos Node portables para build, lint y tests
- El proxy se validó en Windows con compilación, tests unitarios, health check y gestión de proceso
- CI/CD ahora cubre Windows además de Ubuntu

## 🐳 **Despliegue con Docker**

- Nueva imagen Docker multi-stage con runtime no root
- Nuevo `compose.yml` con `restart: unless-stopped`
- Volumen persistente para `cache/`
- Healthcheck nativo sobre `/health`
- Nuevos comandos npm para `docker:build`, `docker:up`, `docker:down`, `docker:logs` y `docker:ps`

## 🚀 **Impacto para usuarios**

- **Windows usable de verdad**: ya no depende de Bash/systemd para operar
- **Autoarranque simple**: con Docker Desktop iniciando con Windows, el contenedor vuelve automáticamente
- **Operación uniforme**: mismo flujo de despliegue en Windows y Linux
- **Mejor mantenibilidad**: documentación y CI alineados con el soporte multiplataforma

## 📦 **Estado validado**

- `npm run build`
- `npm run test:unit` con 103 tests pasando
- `docker compose build`
- `docker compose up`
- `GET /health`
- `POST /v1/chat/completions`
